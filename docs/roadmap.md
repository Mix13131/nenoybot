# Roadmap

## 1. Локальный CLI

- Создать структуру проекта.
- Описать системную инструкцию.
- Добавить боевой словарь и сценарии.
- Реализовать простой keyword-based движок.
- Проверить запуск через терминал.

## 2. Подключение OpenAI / Claude API

- Собрать промпты в единый системный контекст.
- Добавить безопасную конфигурацию API-ключей через `.env`.
- Реализовать режим LLM-ответов.
- Сохранить локальный движок как fallback.

## 3. Telegram-бот

- Добавить Telegram transport-слой.
- Разделить движок и интерфейс.
- Поддержать команды `/goal`, `/report`, `/reset`.

## 4. Хранение целей пользователя

- Добавить локальное хранилище целей.
- Поддержать активную цель, срок и ближайший шаг.
- Добавить простую историю изменений цели.

## 5. История отчетов

- Сохранять отчеты пользователя.
- Показывать прогресс за день и неделю.
- Добавить напоминания и контрольные точки.

## 6. Напоминания

- Добавить локальные или Telegram-напоминания.
- Настроить интервалы контроля.
- Запрашивать отчет после срока.

## 7. Интеграция с Make

- Подготовить webhook-интерфейс.
- Отправлять события цели и отчета в no-code сценарии.
- Поддержать интеграцию с календарем или таск-трекером.

## 8. Дашборд прогресса

- Показать цели, отчеты, пропуски и серии.
- Добавить простую аналитику дисциплины.
- Подготовить веб-интерфейс после проверки CLI.
