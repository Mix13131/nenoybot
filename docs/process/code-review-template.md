# Шаблон Code Review (внутри репо)

## Что проверить

- [ ] Соответствие задаче и AC
- [ ] Нет изменений вне scope
- [ ] Нет новых рисков без фикса
- [ ] Логи/ошибки читаемы
- [ ] Производительность не ухудшилась явно

## Проверить тесты

- [ ] Покрытие добавлено или обновлено
- [ ] Падения отсутствуют

## Комментарии

- [ ] Оставлены только приоритетные замечания
- [ ] Избегать микро-перенастроек без пользы
