# Тест-кейсы

## Prompt Loader

| Сценарий | Ввод | Ожидаемый результат |
| --- | --- | --- |
| Загрузка системного промпта | `system_prompt.md` | Возвращается markdown-текст с ролью НеНойBot |
| Загрузка всех промптов | нет | Возвращается словарь с 4 markdown-файлами |
| Нет файла | `missing.md` | Ошибка `FileNotFoundError` |

## Engine

| Сценарий | Ввод | Цель | Ожидаемый результат |
| --- | --- | --- | --- |
| Цель не указана | `потом сделаю` | пусто | Бот просит цель |
| Прокрастинация | `мне лень` | есть | Бот требует минимальный шаг |
| Сомнение | `сомневаюсь` | есть | Бот предлагает тестовое действие |
| Отчет | `сделал` | есть | Бот принимает отчет и просит следующий шаг |
| Срыв | `сорвался` | есть | Бот возвращает к минимальному действию |
| Отказ | `хочу все бросить` | есть | Бот откладывает эмоциональное решение |
| Перепланирование | `сначала составлю стратегию` | есть | Бот останавливает планирование и требует действие |
| Запрос мотивации | `пни меня` | есть | Бот возвращает к старту без ванильной мотивации |
| Оправдание | `не сделал потому что не было времени` | есть | Бот возвращает ответственность и действие |
| Перегруз | `слишком много задач` | есть | Бот сужает фокус до одной задачи |
| Нет срока | `напишу черновик` | есть | Бот требует конкретное время выполнения |
